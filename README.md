# morning_routine_ML-
Proyek ini adalah aplikasi web yang menampilkan analisis dan klasifikasi tingkat produktivitas berdasarkan data morning routine. Data mencakup jam bangun, durasi olahraga, waktu mandi, sarapan, dan aktivitas pagi lainnya. Web ini dibuat agar visualisasi data, preprocessing, dan hasil klasifikasi dapat diakses publik dengan mudah.
